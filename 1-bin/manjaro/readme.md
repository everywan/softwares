# manjaro
manjaro 相较于其他 arch 发行版更容易上手，且定制性也很强，故日常使用选择此发行版本。其他流行发行版参考 [distrowatch](https://distrowatch.com/?language=CN)

建议使用 manjaro 的官方源，软件更多。比如说 yay 包 manjaro 里有但是 arch 里没有，需要自己编译。
